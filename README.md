## Description

Demo-App for NestJS with TypeORM.  
It is mainly used for testing purposes and to explore and show what could be done with NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
```bash
# watch mode
$ npm run start:dev
```
```bash
# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
```bash
# e2e tests
$ npm run test:e2e
```
```bash
# test coverage
$ npm run test:cov
```
